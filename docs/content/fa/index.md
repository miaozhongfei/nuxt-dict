# Nuxt Dict

Nuxt Dict یک ماژول Nuxt 4 است که به شما کمک می‌کند **داده‌های دیکشنری** (که به آن‌ها جداول کد یا مقادیر شمارشی نیز گفته می‌شود) را در پروژه‌های خود مدیریت کنید.

## داده دیکشنری چیست؟

در توسعه روزمره خود، حتماً با این سناریوها مواجه می‌شوید:

- ستون "جنسیت" در جدول `male` / `female` را نمایش می‌دهد، اما کاربران می‌خواهند "مرد" / "زن" ببینند
- گزینه‌های یک dropdown برای "نوع صنعت" باید از یک API backend دریافت شوند
- یک انتخاب‌گر منطقه به داده‌های ساختار درختی نیاز دارد
- باید یک dropdown یکسان در محیط‌های چینی و انگلیسی متن متفاوتی نشان دهد

**ماژول دیکشنری دقیقاً برای حل این مشکلات طراحی شده است.**

## این ماژول چه کاری می‌تواند انجام دهد؟

- **useDict** — دریافت داده‌های دیکشنری بر اساس نوع، با رابط ترجمه `translate(code)`
- **useDictOptions** — خروجی مستقیم با فرمت `{ label, value }[]`، یکپارچه‌سازی بی‌دردسر با Element Plus / Vant
- **useDictTree** — پشتیبانی از دیکشنری درختی، با `findPath` برای بازگشت مسیر
- **useLocale** — مدیریت چند زبانه، با تشخیص زبان از cookie / header / query
- **کش سه سطحی** — حافظه → IndexedDB → API، با تشخیص خودکار نسخه و ابطال
- **پیش‌واکشی SSR** — دریافت داده‌های دیکشنری در سمت سرور برای رندر سریع اولین صفحه

## شروع سریع (۳ مرحله)

1. **نصب**: `pnpm add @lacqjs/nuxt-dict`
2. **ثبت**: اضافه کردن `'@lacqjs/nuxt-dict'` به آرایه `modules` در `nuxt.config.ts`
3. **استفاده**: نوشتن `const { data, translate } = useDict('gender')` در هر صفحه

## آماده‌اید؟ از [آماده‌سازی محیط](/guide/environment) شروع کنید!
